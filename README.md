## 📂 项目结构  

```plaintext
PaiPai/
├── apps/            # Service
│   ├── im/          # 即时通信服务
│   ├── social/      # 社交服务
│   ├── task/        # 事务服务
│   ├── user/        # 用户服务
│   └── eino_chat/   # EinoService
├── components/      # API网关
├── deploy/          # Docker部署&&运行脚本
├── pkg/             # 工具包
├── test/            # 实例测试等
├── go.mod               
├── docker-compose.yaml   
└── Makefile              
```
